电脑卡顿不用重装系统！完整修复教程（不 Ghost、不重装）

按从简单→深度顺序操作，一步步做，大部分老旧 / 卡顿电脑能恢复 80% 以上速度，全程不用重装系统。
一、先做 3 个立刻见效的简单操作（5 分钟搞定）

1. 关闭开机自启软件（最管用）

Win10/Win11 右键任务栏 → 任务管理器 → 启动应用把所有不需要开机自启的全部禁用：微信、QQ、浏览器、播放器、杀毒软件、各种管家、办公软件等只留显卡驱动、输入法、系统必要项，重启电脑立刻变快。
2. 清理系统临时垃圾

  1. Win+R 输入 %temp% 回车，全选删除（删不掉的跳过）
  2. Win+R 输入 temp 回车，同样全删
  3. 回收站清空
  4. 关闭后台多余程序
    任务管理器→进程结束占用CPU / 内存很高的陌生程序、闲置浏览器标签、没用的后台软件。

二、系统自带优化（不装任何第三方工具）

  1. 磁盘清理
    此电脑 → 系统盘 C 盘 → 右键属性 → 磁盘清理再点清理系统文件勾选：Windows 更新清理、临时文件、缩略图、日志文件等，确定清理。
  2. 关闭虚拟内存 / 视觉特效（低配电脑必做）
  3. 右键此电脑 → 属性 → 高级系统设置 → 高级 → 性能 设置
  4. 勾选调整为最佳性能，也可只保留：平滑字体边缘，其余全关
  5. 再点高级 → 虚拟内存 → 更改把自动管理取消，设置系统盘自定义大小（物理内存 16G 以上可以直接关闭虚拟内存）
  6. 关闭休眠（节省 C 盘空间）
    以管理员身份打开命令提示符，输入：plaintext
    powercfg -h off
    立刻释放几 GB~ 十几 GBC 盘空间。

三、查杀流氓软件、病毒、捆绑后台

  1. 卸载全家桶 / 流氓软件设置 → 应用 → 已安装应用卸载：各种管家、大师、助手、弹窗软件、不知名播放器、推广软件只留一个正规杀毒即可，不要装 2 个杀毒
  2. 全盘查杀用 Windows Defender 或 正规杀毒，全盘扫描木马、恶意插件。

四、修复系统损坏、磁盘坏道（系统变慢常见根源）

  1. 修复系统缺失 / 损坏文件
    管理员 CMD 依次输入两条，每条跑完再输下一条：plaintext
    sfc /scannow
    DISM /Online /Cleanup-Image /RestoreHealth
    自动修复系统底层故障，很多莫名卡顿、闪退都能修好。
  2. 检测修复磁盘错误
    C 盘右键→属性→工具→检查→扫描驱动器有错误自动修复。
  3. 机械硬盘务必磁盘碎片整理
    如果是机械硬盘 HDD：C 盘属性→工具→优化→选中 C 盘优化整理固态硬盘 SSD 不要整理碎片，忽略即可。

五、C 盘爆满变红导致卡顿？无损清理扩容

  1. 把桌面、文档、下载、微信 QQ 缓存迁移到 D/E 盘
  2. 微信电脑端设置→文件管理，改到非 C 盘
  3. 卸载大型软件、游戏装到其他盘
  4. 用分区工具无损把 D 盘空间分给 C 盘（不丢数据、不重装）

六、终极不用重装的提速方案（硬件层面）

如果以上都做了还是很慢，90% 是硬件瓶颈：

  1. 老旧电脑加固态硬盘 SSD，把系统装 SSD，速度直接起飞
  2. 内存小于 8G 的，加到 16G，多开软件再也不卡这两个比折腾系统优化管用十倍，且不用重装也能升级。
